# 📚 BookLens 프로젝트 발표자료

## 1. 프로젝트 개요

### 1.1 프로젝트 소개
**BookLens**는 독서를 즐기는 모든 사람을 위한 종합 독서 관리 플랫폼입니다.

**핵심 가치:**
- 📖 **개인화된 독서 관리**: 나만의 독서 라이브러리 구축 및 진행 상황 추적
- 👥 **커뮤니티 기반 학습**: 독서 후기 공유를 통한 지식 확장
- 🤖 **AI 기반 추천**: 개인 취향에 맞는 도서 추천
- 📊 **데이터 기반 인사이트**: 독서 시간, 완독률 등 통계 제공

### 1.2 프로젝트 목표
- 독서 습관 형성 및 독서 여정 체계적 관리
- 독서 후기 공유를 통한 커뮤니티 활성화
- AI 기술을 활용한 맞춤형 도서 추천
- 외부 API 연동을 통한 편리한 책 정보 검색

## 2. 페이지 구조 및 주요 기능

### 2.1 전체 페이지 구조

```
BookLens
├── 홈페이지 (/)
├── 로그인/회원가입 (/login, /signup)
├── 마이라이브러리 (/mylibrary) 🔒
├── 포스팅 작성 (/posting) 🔒
├── 커뮤니티 (/community)
├── AI 추천 (/recommend)
└── OAuth 콜백 (/auth/callback)
```

**🔒 = 인증 필요 (Protected Route)**

---

### 2.2 페이지별 상세 기능

#### 📍 홈페이지 (HomePage)
**주요 기능:**
- **히어로 섹션**: 웹사이트 소개 및 시작하기 버튼
- **독서 달력**: 날짜별 독서 기록 시각화
  - 날짜 클릭 시 해당 날짜의 상세 독서 기록 확인
  - 독서 시간, 읽은 책, 페이지 수 등 표시
- **현재 읽고 있는 책**: 실시간 독서 세션 표시
  - 독서 시작/종료 기능
  - 실시간 독서 시간 타이머
  - 진행률 시각화

**특징:**
- 인증된 사용자에게만 독서 세션 표시
- OAuth 콜백 처리 (Google, Naver)

---

#### 📚 마이라이브러리 (MyLibraryPage) 🔒
**핵심 기능:**

1. **책 추가**
   - **ISBN 검색**: ISBN 입력으로 Google Books API에서 책 정보 자동 불러오기
   - **제목/저자 검색**: 실시간 검색 (디바운싱 500ms 적용)
   - **수동 입력**: 직접 책 정보 입력 가능
   - 책 표지 미리보기

2. **독서 진행 관리**
   - **독서 시작/종료**: 모달을 통한 세션 관리
   - **실시간 타이머**: 1초마다 업데이트되는 독서 시간 측정
   - **페이지 입력**: 읽은 페이지 수 입력 및 진행률 자동 계산
   - **상태 관리**: 읽는 중 / 완독 / 미시작 상태 추적
   - **독서 기록 저장**: 날짜별 독서 기록 자동 저장 (달력용)

3. **통계 대시보드**
   - 전체 도서 수
   - 읽는 중인 책 수
   - 완독한 책 수
   - 총 독서 시간
   - 오늘 읽은 시간
   - 내 포스팅 개수

4. **완독 책 관리**
   - 완독한 책 목록 별도 섹션 표시
   - 완독일 기록
   - 포스팅 작성 바로가기

5. **독서 페르소나 배지**
   - 독서 패턴 분석 기반 페르소나 표시
   - 장르, 독서 시간, 완독률 등 기반 계산

**UI/UX 특징:**
- 카드 기반 레이아웃
- 진행률 프로그레스 바
- 실시간 시간 표시 (진행 중 세션 포함)
- 모달을 통한 독서 세션 관리
- 반응형 디자인 (모바일, 태블릿, 데스크톱)

---

#### ✍️ 포스팅 작성 (PostingPage) 🔒
**주요 기능:**
- **포스팅 작성**
  - 제목 입력 (선택사항, 기본값: "책제목 독후감")
  - 평점 선택 (별점 1-5점)
  - 내용 작성 (필수, 실시간 글자 수 표시)
  - 태그 입력 (쉼표로 구분)
  
- **포스팅 수정**
  - 기존 포스팅 수정 모드 지원
  - 기존 내용 자동 불러오기
  - 수정 후 커뮤니티로 이동

- **책 정보 표시**
  - 완독한 책 정보 카드
  - 책 제목, 저자, 완독일 표시

**특징:**
- 완독한 책만 포스팅 작성 가능
- 직관적인 폼 UI
- 실시간 글자 수 표시

---

#### 👥 커뮤니티 (CommunityPage)
**핵심 기능:**

1. **포스팅 목록**
   - 모든 사용자의 독서 후기 표시
   - 그리드 레이아웃 (반응형: 1열/2열/3열)
   - 책 정보, 평점, 태그, 작성자 정보 포함
   - 내용 미리보기 (150자 제한)

2. **검색 및 필터링**
   - **검색**: 책 제목, 저자, 내용, 태그로 검색
   - **정렬**: 최신순, 평점순, 오래된순
   - **필터**: 내 포스트만 보기 토글

3. **포스팅 상세 정보**
   - 제목, 내용 미리보기
   - 평점 (별점 1-5점 시각화)
   - 태그 표시 (최대 3개, 나머지는 +N 표시)
   - 작성 시간 표시 (상대 시간: "방금 전", "3시간 전", "2일 전" 등)
   - 완독일 정보

4. **포스팅 관리**
   - 내 포스팅 수정 (수정 버튼)
   - 내 포스팅 삭제 (삭제 버튼)
   - 자세히 보기 버튼

**특징:**
- 반응형 3열 그리드 레이아웃
- 실시간 검색 (디바운싱 적용)
- localStorage 기반 데이터 저장
- 날짜 포맷팅 (상대 시간 표시)

---

#### 🤖 AI 추천 (RecommendPage)
**주요 기능:**
- **AI 도서 추천**
  - 책 제목 입력
  - 좋아하는 장르 입력 (선택, 쉼표로 구분)
  - AI 추천 결과 표시
    - 기준 도서 정보 (제목, 저자, 장르, ISBN)
    - 추천 도서 목록 (제목, 저자, 장르, 추천 이유, 키워드)

**특징:**
- 백엔드 AI API 연동 (`/api/ai/recommendations`)
- 사용자 컨텍스트 기반 추천 (선호 장르 등)
- 추천 이유 및 키워드 제공

---

#### 🔐 로그인/회원가입 (LoginPage / SignupPage)
**주요 기능:**

**로그인:**
- 이메일/비밀번호 로그인
- 실시간 폼 검증 (이메일 형식, 비밀번호 길이)
- 로그인 상태 유지 옵션
- Google OAuth 로그인 버튼
- 비밀번호 찾기 링크

**회원가입:**
- 이메일/비밀번호/이름 회원가입
- 실시간 폼 검증:
  - 이메일 형식 검증
  - 이름 검증 (2-20자, 한글/영문/숫자만)
  - 비밀번호 검증 (최소 6자)
  - 비밀번호 확인 일치 검증
- Google OAuth 회원가입 버튼

**특징:**
- 깔끔한 카드 디자인
- 실시간 에러 메시지 표시
- 폼 유효성 검사

## 4. 기술 스택

### 4.1 Frontend
- **React 19.2.0**: 사용자 인터페이스 구축
- **Vite 7.2.4**: 빠른 개발 환경 및 빌드 도구
- **React Router DOM 7.10.0**: 페이지 라우팅
- **Tailwind CSS 3.4.18**: 유틸리티 기반 스타일링
- **Context API**: 전역 상태 관리 (AuthContext, FocusSoundContext)

### 4.2 Backend
- **Node.js + Express**: REST API 서버
- **PostgreSQL / Supabase**: 관계형 데이터베이스
- **JWT**: 인증 토큰
- **bcrypt**: 비밀번호 해싱

### 4.3 External APIs
- **Google Books API**: 책 정보 검색 및 상세 정보
- **Supabase**: 인증 및 데이터베이스
- **백엔드 REST API**: 독서 관리, 커뮤니티, AI 추천

### 4.4 데이터 저장
- **Supabase**: 서버 사이드 데이터 저장
- **localStorage**: 클라이언트 사이드 캐시 (폴백)
- **sessionStorage**: 인증 토큰 저장
