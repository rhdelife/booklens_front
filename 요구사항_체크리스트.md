## 2️⃣ 백엔드

### ⚠️ REST API 최소 4개 이상
- **상태**: ⚠️ **프론트엔드에서 API 정의는 확인됨, 실제 백엔드 서버 코드는 현재 워크스페이스에 없음**
- **프론트엔드 API 정의** (28개 이상):
  - **인증 API** (`/api/auth/*`): 8개
    - POST `/api/auth/login`
    - POST `/api/auth/signup`
    - POST `/api/auth/logout`
    - GET `/api/auth/me`
    - PUT `/api/auth/profile`
    - POST `/api/auth/profile/image`
    - POST `/api/auth/google/callback`
    - POST `/api/auth/naver/callback`
  
  - **책 API** (`/api/books/*`): 5개
    - GET `/api/books`
    - POST `/api/books`
    - GET `/api/books/:id`
    - PUT `/api/books/:id`
    - DELETE `/api/books/:id`
  
  - **독서 세션 API** (`/api/reading-sessions/*`): 6개
    - POST `/api/reading-sessions`
    - GET `/api/reading-sessions/active`
    - PUT `/api/reading-sessions/:id`
    - POST `/api/reading-sessions/save`
    - GET `/api/reading-sessions/calendar`
    - GET `/api/reading-sessions/date`
  
  - **포스팅 API** (`/api/postings/*`): 5개
    - GET `/api/postings`
    - POST `/api/postings`
    - GET `/api/postings/:id`
    - PUT `/api/postings/:id`
    - DELETE `/api/postings/:id`
  
  - **좋아요 API**: 1개
    - POST `/api/postings/:id/like`
  
  - **댓글 API**: 2개
    - POST `/api/postings/:id/comments`
    - DELETE `/api/comments/:id`
  
  - **AI 추천 API**: 1개
    - POST `/api/ai/recommendations`

- **백엔드 설계 문서**: `BACKEND_DESIGN_PROMPT.md`에 상세한 API 스펙 정의
- **프로젝트 요약**: `PROJECT_SUMMARY.md`에 백엔드 구조 언급 (`booklens2-backend/` 디렉토리)
- **⚠️ 주의**: 실제 백엔드 서버 코드는 현재 워크스페이스에 없어 실제 구현 여부 확인 불가
- **권장사항**: 백엔드 서버가 별도 디렉토리(`booklens2-backend/`)에 있다면 해당 디렉토리도 확인 필요

### ✅ 요청/응답 JSON 처리
- **상태**: ✅ **충족** (프론트엔드 코드 기준)
- **증거**:
  - `src/services/api.js`의 `apiRequest` 함수에서:
    - `Content-Type: application/json` 헤더 설정
    - `JSON.stringify()`로 요청 본문 변환
    - `response.json()`으로 응답 파싱
    - JSON 형식 에러 응답 처리 (`{ error: "..." }`)

  - Update: `updateBook`, `updatePosting`, `updateProfile` 등
